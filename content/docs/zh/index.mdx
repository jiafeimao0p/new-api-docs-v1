## 欢迎使用 1API 中转站

**1API** 是一个基于 New API 构建的 AI 模型中转服务，支持将各类大语言模型聚合管理，并以 OpenAI、Claude、Gemini 兼容接口对外分发。

## 支持的模型类型

| 类型 | 说明 |
| --- | --- |
| 聊天补全 | GPT-4o、Claude 3.5、Gemini 1.5、DeepSeek V3、Qwen 等 |
| 向量嵌入 | text-embedding-3、voyage 系列等 |
| 图像生成 | DALL-E、Flux、Midjourney、即梦、可灵等 |
| 语音处理 | Whisper、TTS、Audio 等 |
| 实时对话 | OpenAI Realtime API 兼容 |
